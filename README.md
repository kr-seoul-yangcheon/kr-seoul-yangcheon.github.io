# 양천구 · 서울특별시

> 양천구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-yangcheon.github.io/**

양천구는 목동 신시가지라는 대한민국 대표 계획도시와 목동 학원가로 상징되는, 서울에서 교육열이 가장 뜨거운 자치구입니다. 1988년 강서구에서 갈라져 나올 때 신정동에서 딴 '신정구' 명칭과 경합한 끝에 옛 고을 이름 '양천'을 되살려 지금의 구명을 갖게 되었습니다. 목동서로에 자리한 SBS 방송센터를 비롯해 방송·미디어 관련 기관이 밀집해 방송 거점으로도 이름이 높습니다. 안양천을 사이에 두고 영등포구와 마주하며, 17.41㎢의 좁은 면적에 40만 명이 넘는 인구가 살아 전국 기초자치단체 가운데 인구밀도가 가장 높은 곳으로 꼽힙니다. 한-프랑스 수교 100주년을 기념한 파리공원과 옛 정수장을 되살린 서서울호수공원처럼 이야기가 담긴 녹지도 곳곳에 배치돼 있습니다. 아파트 숲과 학원가, 방송단지, 재생 공원이 어우러진 도심형 주거·교육 도시가 양천구의 얼굴입니다.
